**核心功能 - 滚动透明度计算:**
```typescript
const scrollY = ref(0)

// Header 整体透明度 (50px-200px 范围渐变)
const headerOpacity = computed(() => {
  const start = 50
  const end = 200
  if (scrollY.value <= start) return 1
  if (scrollY.value >= end) return 0
  return 1 - (scrollY.value - start) / (end - start)
})

// 背景透明度 (100px-300px 范围渐变)
const bgOpacity = computed(() => {
  const start = 100
  const end = 300
  if (scrollY.value <= start) return 1
  if (scrollY.value >= end) return 0.3
  return 1 - (scrollY.value - start) / (end - start) * 0.7
})

// 监听滚动
onMounted(() => {
  window.addEventListener('scroll', handleScroll, { passive: true })
})
```

## 技术要点

### 1. 滚动透明效果原理
```
滚动位置: 0px
  → Header 100% 可见
  → 背景 100% 不透明

滚动位置: 50px
  → Header 开始淡出

滚动位置: 100px
  → Header 约 75% 可见
  → 背景开始变透明

滚动位置: 200px
  → Header 完全透明（不可见）

滚动位置: 300px
  → 背景 30% 透明度（仍可见毛玻璃效果）
```

### 2. 毛玻璃效果实现
```css
.app-header {
  backdrop-filter: blur(12px);
  background: rgba(var(--color-bg-secondary-rgb), var(--bg-opacity));
}
```

**关键点:**
- 使用 `backdrop-filter: blur()` 实现毛玻璃
- 需要 RGB 格式的背景色变量（非 hex）以实现透明度控制
- `--color-bg-secondary-rgb: '26, 26, 26'` 对应 rgba(26, 26, 26, var(--bg-opacity))

### 3. 响应式断点设计
| 断点 | 目标设备 | 变化 |
|------|----------|------|
| >768px | 桌面端 | 完整显示所有元素 |
| ≤768px | 平板 | 隐藏 Tab，保留 Logo 和用户 |
| ≤480px | 手机 | 额外隐藏搜索框，只保留 Logo 和用户 |

### 4. 硬编码颜色修复
修复前的硬编码颜色:
```css
/* ❌ 修复前 */
.featured-overlay {
  background: rgba(255, 255, 255, 0.9); /* 硬编码白色 */
}
```

修复后使用 CSS 变量:
```css
/* ✅ 修复后 */
.featured-overlay {
  background: linear-gradient(
    135deg,
    var(--color-bg-secondary),
    var(--color-bg-tertiary)
  );
  color: var(--color-text-primary);
}
```

## 备注
Header 组件的设计考虑:
1. **性能**: 使用 CSS transform 和 opacity 实现动画，避免重排
2. **体验**: 滚动时不完全隐藏，保留 30% 透明度以便用户知道 Header 位置
3. **无障碍**: 保持 Tab 焦点状态，支持键盘导航
